# Fullstack JavaScript MERN App

Simple Note-app made in stack MERN, you can create, delete, edit and get notes and users.

## Technologies

## Frontend

* [React](https://reactjs.org/)
* [react-router-dom](https://www.npmjs.com/package/react-router-dom)
* [Bootstrap](https://getbootstrap.com/)
* [axios](https://www.npmjs.com/package/axios)
* [timeago.js](https://www.npmjs.com/package/timeago.js/v/4.0.0-beta.3)
* [react-datepicker](https://reactdatepicker.com/)

### Backend

* [NodeJS](https://nodejs.org/en/)
* [ExpressJS](https://expressjs.com/)
* [Mongoose](https://mongoosejs.com/)
* [dotenv](https://www.npmjs.com/package/dotenv)
* [cors](https://www.npmjs.com/package/cors)

## Utils

* [VScode](https://code.visualstudio.com/)
* [Chrome](https://www.google.com/intl/es-419/chrome/)
* [Insomnia](https://insomnia.rest/)
